STM32F429 Discovery Board Firmware Application Package Update  History

  

The STM32F429 DISCOVERY Board Firmware Applications Package provides ready-to-run firmware examples to support quick evaluation and development on STM32F429I Discovery board (STM32F429I-DISCO). It contains the following examples:
  • Upgrading STM32F429  DISCOVERY board firmware using a USB key, under Project\FW_upgrade folder, for more details about this application refer to the associated document AN3990
  • STM32F429 DISCOVERY peripheral firmware examples, under Project\Peripheral_Examples, for more details about each application refer to associated readme file
  • STM32F429 DISCOVERY Board Demonstration firmware, under Project\Demonstration, for more details about this application and the overall package structure refer to the associated document "Getting started with the STM32F429 Discovery kit (UM1662)"

For more information on the STM32F429 Discovery, to download the available firmware package and documentation visit www.st.com/stm32f4-discovery.
